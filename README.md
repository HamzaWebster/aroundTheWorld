## 📖 Project Concept
**Around the World** is a digital media production project created for the **Introduction to Media Production** course. Our objective was to move beyond traditional single-medium assignments (like standalone videos or podcasts) and instead create a **convergent media platform**. 

By integrating cinematic video, ambient audio, and structured web narratives, we aimed to reflect how modern media organizations actually operate—where content is not just produced, but carefully organized and presented within a custom digital ecosystem.

## 🚀 Key Production Philosophies
- **Custom-Built from Scratch**: We intentionally avoided ready-made platforms like WordPress to understand the fundamental building blocks of digital media—how files are structured, how media is linked, and how the user experience is crafted from the ground up.
- **Immersive Multi-Medium Experience**: The project explores the synergy between different media types. Watching a video inside a custom-designed page with ambient audio and descriptive text provides a deeper level of engagement than a video on its own.
- **Agentic AI Workflow**: Instead of using AI as a simple chatbot, we integrated it as a **professional production agent**. By establishing fixed rules, shared skills, and design constants within the repository, we created a structured system that ensured visual consistency and architectural integrity throughout the development process.

## 📂 Project Architecture
The project is organized into a modular structure to ensure maintainability and professional standards:
```bash
├── .agents/                # The "Brain" of the production
│   ├── rules/              # Production standards & development laws
│   ├── skills/             # Instructions for building complex UI components
│   └── theme/              # Centralized Design Tokens (Colors, Typography)
├── index.html              # The digital hub and team gateway
├── cairo.html              # Curated experience: Cairo, Egypt
├── mani.html               # Curated experience: Mani, Greece
├── privacy.html            # Curation policy & intellectual property rights
└── README.md               # Production documentation
```

## 🛠️ Technical Implementation
- **Frontend**: Custom HTML5 and Vanilla CSS3.
- **Framework**: Bootstrap 5.3 (Customized via Design Tokens).
- **Hosting**: Deployed via GitHub Pages for high-performance, static delivery.
- **AI Integration**: Orchestrated via an Agentic Workflow to maintain "Single Source of Truth" for design and code patterns.

## 🖋️ Final Reflection
This project was not just about building a website; it was an exercise in teamwork, media convergence, and modern production technology. It taught us that production is as much about **organization and presentation** as it is about the content itself.
